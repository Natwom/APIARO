# APIARO - Full-Stack E-Commerce Platform

A production-ready e-commerce platform tailored for Kenya with M-Pesa integration, SMS notifications, and multi-county delivery support.

## Features

- **User Features**: Registration/Login (JWT), Product browsing, Search, Cart, Checkout
- **Payment Methods**: Cash on Delivery, M-Pesa (simulated), Card Payment (simulated)
- **SMS Notifications**: Automated SMS on order placement (Mock service ready for Africa's Talking)
- **Admin Panel**: Order management, Status tracking, Customer phone number visibility
- **Kenya-Specific**: 47 Counties support, Kenyan phone format (+254)
